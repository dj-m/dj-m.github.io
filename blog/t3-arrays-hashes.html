Arrays & Hashes... for now

June 24, 2015

So here’s the problem… you want to store some data that you can access for specific situations, but there’s two ways to do this (in programming). Huh?

I know, it’s still pretty shaky to me, but here’s where I’m at:

One method of collecting this data is in an array. An array is a list much like you’d make for grocery shopping. Let’s say you have soymilk, lettuce and almonds on your shopping list. You can access or look up the values of your list based on where in that list your grocery item is located.

So, your array, let’s call it ‘shopping_array’ has 3 items. You may want to know what the first item in the list is and using shopping_array[0] would give you “soymilk.” The second item would be “lettuce” using shopping_array[1].

You may have noticed that the first item in an array is referenced by 0 rather than 1. It’s super important to remember that this will usually be the case for any lists. Counting starts from 0.

Now a hash is a list, but of paired items. Each pair has a name and a value. The name, or ‘key’, must be unique to each value paired with it.

So, for instance, if soymilk was on your grocery list, then ‘vanilla’ or ‘coffee’ flavor would be next to that. In this example ‘soymilk’ is the key for ‘vanilla’. Instead of a list, think of this structure as more akin to a table than a simpler grocery list.

What’s important with hashes isn’t so much the order, but what’s paired with the values.

Examples:

Your grocery list array might look like so:

shopping_list = [“soymilk”, “lettuce”, “almonds”]

A grocery hash may look like so:

specific_list = [“soymilk” => “low-fat”, “lettuce” => “romaine”, “almonds” => “unsalted”]

In this case, the “soymilk”, “lettuce” and “almonds” would be your key and the “low-fat”, “romaine” and “unsalted” would be your value, respectively.

Lookups:

To lookup information from your array, you would type the following:

shopping_list[2] and get “almonds”

While in a hash, you’d type the following:

specific_list[“almonds”] and get “unsalted”
