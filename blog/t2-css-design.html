CSS Concepts... for now

June 18, 2015

What are the best practices associated with using classes vs. ids?

CSS classes are a group of elements that are similar and have attributes, like color and font sizes, that are specific.

To add a class to your HTML type the following:

< div class="class" >

To use a class in your CSS type the following:

.class {
width:50px;
height:150px;
background:lavender
}

Keep in mind, classes are NOT unique

  • You can use the same class on multiple elements
  • You can use multiple classes on the same elements

An ID, by contrast, singularly identifies one HTML tag. You can have one HTML tag per ID, and each one has a specific set of CSS attributed that only apply to that element

To add an ID to your HTML type the following:

< div id="id" >

To use an ID in your CSS type the following:

#id {
width:50px;
height:150px;
background:lavender

Keep in mind, IDs are unique

  • Each element can only have one ID
  • Each page can have only one element with that ID
